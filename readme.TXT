Einfache Verwaltung der monatlichen Blogeinnahmen und Darstellung der Daten in verschiedenen Diagrammen.

== Description ==

In der Tabelle können alle Blogeinnahmen einfach im Adminbereich verwaltet werden. Es können beliebig viele Einnahmequellen und Monate hinzugefügt werden.
Die Gesamteinnahmen und eCPM werden automatisch berechnet. Außerdem lassen sich die gesamten Daten der Tabelle als .csv-Datei exportieren.

Anschließend können die Daten wahlweise in einem Balkendiagramm oder einem Kreisdiagramm grafisch dargestellt werden. Dabei kann man einen bestimmten Monat oder längere Zeiträume ausgewählen.
Mithilfe eines Shortcodes kann man die Diagramme auch einfach in seinem Blogartikel darstellen.

Bei Fragen, Wünschen und Verbesserungsvorschlägen bitte die folgende Seite besuchen: http://www.allblogs.de/wordpress-plugin-blogeinnahmen/wordpress/plugins/2012/06/

== Installation ==

1. Den Ordner 'Blogeinnahmen' in das Verzeichnis '/wp-content/plugins/' hochladen
2. Das Plugin im 'Plugins' Menü aktivieren
